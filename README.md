## 🎯 О проекте и цель разработки
Этот проект — мой персональный **Pet-проект**, представляющий собой многостраничный веб-сайт для концептуального магазина оптики. В отличие от простой верстки по чужому макету, здесь я реализовал **полный цикл разработки**: самостоятельно разработал визуальный UI/UX дизайн интерфейса, а затем воплотил его в чистом коде с нуля.

Главная цель — глубокая отработка навыков создания интерфейсов, построения сложных сеточных структур, оптимизации графики под Retina-дисплеи и обеспечения безупречной адаптивности без использования тяжелых UI-фреймворков.

## 🛠️ Какие навыки я прокачал на этом проекте
*   **UI/UX Проектирование:** Прошел путь от идеи до готовой дизайн-системы, проработав композицию, типографику и пользовательские сценарии.
*   **Pixel-Perfect верстка:** Добился идеального соответствия написанного кода собственной дизайн-задумке.
*   **Модульная архитектура CSS:** Разделил стили на логические блоки (`reset.css` для сброса, `style.css` для основных компонентов, `media.css` для адаптива) для чистоты и удобства поддержки кода[cite: 2].
*   **Продвинутая адаптивность (Responsive Design):** Написал гибкие медиа-запросы, благодаря которым интерфейс идеально подстраивается под любые экраны — от мобильных телефонов до широкоформатных мониторов[cite: 2].

## 💻 Технологический стек
*   **HTML5** (Семантическая структура и правильная иерархия тегов)
*   **CSS3** (Flexbox и Grid Layout для построения сеток[cite: 2], Custom Properties для управления дизайн-системой, Media Queries[cite: 2])
*   **Графика:** Оптимизированные SVG-иконки для сохранения максимальной четкости[cite: 2].

## 🚀 Инструкция по локальному запуску

Для корректного отображения всех страниц и стилей проект рекомендуется запускать через локальный веб-сервер. Выберите любой удобный способ:

### 1. Подготовка (Клонирование репозитория)
Откройте терминал и выполните команды:
```bash
# Клонируем проект
git clone [https://github.com/BibaPirat6/ShopOptic-site.git](https://github.com/BibaPirat6/ShopOptic-site.git)

# Переходим в папку проекта
cd ShopOptic-site
```

### 2. Запуск через Visual Studio Code (Live Server)

Если вы работаете в VS Code[cite: 2]:
* **Установите расширение Live Server** (автор: *Ritwick Dey*) через вкладку Extensions (`Ctrl+Shift+X`)[cite: 2].
* **Откройте папку проекта** в редакторе[cite: 2].
* **Кликните правой кнопкой мыши** по файлу `index.html` и выберите **"Open with Live Server"**[cite: 2].
* **Сайт автоматически откроется** в браузере по адресу `http://127.0.0.1:5500`[cite: 2].
